Website's Theme

  • A colourful abstract background was chosen because it includes a combination of my favourite colours,
    its colour scheme matches the picture used as the home page background and it does not clash with the
    black or blue texts nor images on the website

  • Moving headings were used on some pages to add a little spice to the webpage design.


Webpages and their Purpose

  • Home Page - Welcomes the user to the website.

  • Personal Page - Tells the user about my personality, likes, hobbies and my motto.

    • Photography Page - Displays a few of the pictures I have taken and edited.

  • Professional Page - Provides my resume, which includes a mission and vision statement, along with a list of my skills, experiences and community service.

  • Personal Development Page - Speacks about my strengths and development and summarizes the Twelve Pillars.

  • Reference Page - A page that includes all the bibliographic sources of information for assistance i.e. books and websites in APA format.

  • Story Board Page - Contains an overview of the website, a graphic layout and a description of each page.



Target Audience

  • The website is targeted at people who would like to know more about Nathan.
